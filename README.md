# 🧠 AI MCQ Generator Pro

An intelligent **Generative AI-powered MCQ Generator** that automatically creates high-quality multiple-choice questions from various input formats including PDFs, documents, images.

The application leverages **Large Language Models (LLMs), Natural Language Processing, OCR, and Speech Processing** to generate customized quizzes with difficulty levels and Bloom's Taxonomy-based learning assessment.
